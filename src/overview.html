The smart working test application core functionality allows office employees to request smart working (i.e. work from home) to the office manager. This is a test web application using JSP and servlet technology. Back end database used during development was MySql server. Don't run this application in production environment as it was developed for training purposes only.

The application was developed with jdk 1.8.0_65 and Tomcat application server 8.0.
